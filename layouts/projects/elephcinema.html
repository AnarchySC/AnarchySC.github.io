// pipeline.flow

Three steps. Fully automated.

Insert a disc and the udev rule fires. MakeMKV rips every title to local storage. A classification step identifies Movie vs TV, you confirm via a quick dialog, then ffmpeg VAAPI encodes everything in parallel. The finished MP4 is validated and copied to your NAS. The disc ejects. You never touched the keyboard.

INSERT DISC
→
UDEV TRIGGER
→
MAKEMKV RIP ALL
EJECT + NAS COPY
←
VAAPI ENCODE
←
CLASSIFY + DIALOG

// tech.specs

Built on battle-tested Linux tools.

Ripping
MakeMKV

LibreDrive-compatible. Single invocation rips all titles. Supports 4K UHD, Blu-ray, and DVD.

Encoding
ffmpeg VAAPI

Hardware HEVC on AMD (hevc_vaapi). CQP mode: QP 24 for Blu-ray, QP 20 for DVD. Audio stream copy.

Automation
udev + systemd

Disc insertion triggers the full pipeline via udev rules. Cooldown prevents re-triggers. Runs as your user.
