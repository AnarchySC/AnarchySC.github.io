Organization website for AnarchyGames — built with Hugo. Blade Runner cyberpunk aesthetic with neon glows, holographic effects, and CRT scanlines.

## Projects

- **[ELEPHCINEMA](https://github.com/AnarchySC/elephcinema)** — Automated Blu-ray/DVD ripping pipeline
- **[Oblysk](https://github.com/AnarchySC/oblysk)** — Smart clipboard manager for virtual consoles
- **[Gall-array](https://github.com/AnarchySC/galleray)** — Minimalist image gallery for Linux
- **[In the Black](https://github.com/AnarchySC/in-the-black-game-anarchygames)** — Pong-variant space combat game
- **[Skilltwee](https://github.com/AnarchySC/Skilltwee-AZ104)** — Gamified learning skill trees
- **[Cassette Future Magazine](https://github.com/AnarchySC/magazine)** — Retro-futuristic publication

## Development
